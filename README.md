# Stream RPG Overlay

配信をゲーム化するためのOBS向けRPGオーバーレイツールです。

クエスト達成で経験値を獲得し、レベルアップを目指します。

REALITYやOBSを利用した雑談配信・企画配信・耐久配信での利用を想定しています。

---

## Features

* クエスト管理
* 経験値システム
* レベルアップシステム
* クエスト達成演出
* レベルアップ演出
* OBS対応
* サーバー不要
* ブラウザのみで動作

## How It Works

配信中に発生した出来事をクエストとして管理します。

例:

* 初見さんが来る
* ギフトをもらう
* 1時間配信する
* フォローを獲得する

クエストを達成すると経験値を獲得し、一定値に達するとレベルアップします。

リスナーには現在の目標や進捗が表示されるため、配信への参加感を高めることができます。

---

## Files

### control.html

配信者用の管理画面です。

機能:

* クエスト追加
* クエスト削除
* クエスト達成
* 経験値追加
* データリセット

### overlay.html

OBSに表示するオーバーレイ画面です。

表示内容:

* 現在の目標
* クエスト進捗
* レベル

また、クエスト達成時やレベルアップ時に演出を表示します。
低確率であたり演出とはずれ演出が発生します。
こちらはoverlay.htmlに表示される閉じるボタンをクリックしないと非表示になりません。

---

## Usage

1. control.html を開く
2. overlay.html を別ウィンドウで開く（control.htmlと同じブラウザを使うこと）
3. OBSで overlay.html をキャプチャする
4. control.html から操作する

---

## Storage

進行状況はブラウザの LocalStorage に保存されます。

サーバーやデータベースは不要です。
